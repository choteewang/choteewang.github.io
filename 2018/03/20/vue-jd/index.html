基于Vue,Hotcss+Px2rem自适应设计,Webpack+Npm script自动化构建的jd-finance web app

2018年3月20日

A jd-finance web app based on Vue, hotcss + px2rem

Tech

  • Vue 2.5+
  • Vue-router
  • sass
  • css-modules
  • hotcss
  • px2rem
  • full-ES6-syntax-cover
  • eslint
  • vue-awesome-swiper
  • webpack
  • npm scirpts

自适应方案设计

  • 利用hotcss动态调整meta标签和dpr, 动态设定html标签的font-size(Rem布局实现)
  • 利用px2rem-loader在自动化构建时自动转换scss的css尺寸
  • 一份代码, 所有移动设备动态适配

详情请移步我的技术博客: 记一次vue项目的自适应方案设计 hotcss + px2rem

自动化构建

使用webpack与npm script进行自动化构建, 手写所有代码

  • 处理vue,js,css,scss
  • 处理图片,开启Base64
  • 自动清理dist目录
  • 热重载
  • 提取vendors与manifest
  • Js-Uglify
  • Css-Minify
  • 区分开发环境和生产环境
  • 生产环境提取css文件
  • 开发环境开启js-SourceMap
  • 开发环境开启css-SourceMap
  • eslint

css模块化

使用css模块化, 对公用模板组件的css进行抽象, 后续开发完全继承于已抽象的css模块

vue组件模块化

抽象了slider, btn, panel 三个js组件模块, 后续所有组件开发继承于已抽象组件
